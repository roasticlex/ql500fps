# ql500fps
Quake Live 500fps patch
- while Quake Live is running, open QuakeLive500FPS.exe (under bin folder)
- if you would like to compile yourself you need Visual Studio (I used VS2015), .NET 4.8 developer pack as well as the MemorySharp package (you can add it using Nuget package manager in VS)
